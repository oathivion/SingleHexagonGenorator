# Single Hexagon Generator

A dependency-free browser application for creating a custom flat-top hexagon.

## Features

- Exact width and height from 16 to 4096 pixels
- Six-digit hex color validation
- Live transparent preview
- Transparent PNG export
- High-quality JPG export with white corners (JPG does not support transparency)
- Responsive and keyboard-accessible interface
- Object-oriented architecture separating geometry, validation, rendering, export,
  and application control

## Run

Open `index.html` in a modern browser, or serve the folder with any static web
server.

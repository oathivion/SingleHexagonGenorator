# Polygon Generator

A dependency-free browser application for creating custom polygons from three to
twenty sides.

## Features

- Triangle through icosagon shape selection
- 2D and interactive 3D rendering modes
- Mouse, touch, and keyboard rotation in the 3D viewer
- Classic tetrahedron, cube, and dodecahedron forms for 3–5
- Fifteen individually designed showcase solids for 6–20
- Every custom solid prominently includes its selected polygon as a face
- Crystal, crown, lantern, turbine, vault, and other unique silhouettes
- Shaded closed meshes with perspective projection
- Exact width and height from 16 to 4096 pixels
- Six-digit hex color validation
- Live transparent preview
- Transparent PNG export
- High-quality JPG export with white corners (JPG does not support transparency)
- Responsive and keyboard-accessible interface
- Reusable normalized-vertex algorithm for every supported polygon
- Object-oriented architecture separating geometry, 2D/3D rendering,
  interaction, validation, export, and application control

## Run

Open `index.html` in a modern browser, or serve the folder with any static web
server.
